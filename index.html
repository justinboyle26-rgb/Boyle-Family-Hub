📋 Google Apps Script Setup

Family Hub uses Google Sheets as its database via a simple Apps Script web app:

  1. Create a new Google Sheet — name it "Family Hub"
  2. Open Extensions → Apps Script
  3. Delete any existing code and paste the script below
  4. Click Deploy → New Deployment → Web App
  5. Set Execute as: Me & Who has access: Anyone
  6. Copy the Web App URL and paste it in the setup screen
